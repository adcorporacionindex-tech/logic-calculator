# 🧮 Calculadora

Calculadora web moderna con operaciones básicas + diseño glassmorphism.

### ✨ Features
- **Operaciones:** Suma, resta, multiplicación, división
- **UI Glassmorphism:** Efecto vidrio + degradados modernos
- **Responsive:** Se adapta a móvil y desktop
- **Bug-Free:** 6 bugs cazados y corregidos v1.0

### 🛠️ Tech Stack
`HTML5` `CSS3` `JavaScript Vanilla` `DOM Manipulation`

### 💡 Cómo usar
1. Click en los números para operar
2. Usa `C` para limpiar todo
3. `=` para obtener resultado

### 🐛 Bugs Corregidos v1.0
- Botón `=` sin funcionalidad
- `C` no limpiaba display 
- Operaciones consecutivas fallaban
- Pantalla se desbordaba
- CSS no cargaba rutas
- Validación de decimales
